An appointment booking theme for odoo

This is "Theme Viver" an elegant and stunning theme for appointment booking. Theme Viver specially made for appointmnent booking such as salon, spa, clinic and many more.

Theme Viver designs are based on "Less Is More" principle which means simplicity and cleanliness, and these are the keys points of the theme. It provides easy UI/UX experience to book an appointment so that we can reduce our waiting time at many places such as salons, clinics.
At the same time, it is very easy to set up because all snippets have black and white color combination which complement the whole design of the theme. This way you do not have to bother about the designing, and other things. Anyone without any qualification of designer or developer can setup the theme without any difficulty.
Just try once and you will be amazed by the response because we believe in "not only try but try our best".
